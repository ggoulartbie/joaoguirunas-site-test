# themaestrisites

Dev squad otimizada para criação de sites, landing pages e páginas web de alta conversão. Orquestração multi-terminal via Maestri CLI. Tema: personagens do Mahabharata.

## Squad Composition (8 agents)

| Agent | ID | Icon | Role |
|-------|-----|------|------|
| **Hanuman** | chief | 🐒🔱 | Orchestrator — Maestri CLI |
| **Arjuna** | ux-alpha | 🏹🎨 | UI Component Designer + Dev Front (21st dev) |
| **Nakula** | ux-beta | ✨🐴 | UX/Interaction + Layout + Dev Front (21st dev) |
| **Draupadi** | ux-gamma | 👑🖼️ | Image Creator + Visual Identity + Dev Front |
| **Kunti** | ux-delta | 🙏🌺 | UX Research + Accessibility + Dev Front |
| **Yudhishthira** | copywriter | 👑✍️ | Copywriter — LP sections |
| **Sahadeva** | seo-analyst | 🔍⭐ | SEO Analyst — Technical & On-Page |
| **Bhima** | dev-devops | 💪⚡ | Integration + Deploy (EXCLUSIVE push) |

## Key Differentiators

- **4 UX/Dev Front** that IMPLEMENT code (React/Tailwind/Next.js), not just design
- **21st dev tools** for component creation (builder, inspiration, refiner)
- **Freepik** for image generation (Draupadi)
- **Bhima = Dev + DevOps** combined (squad enxuta for sites)
- **Full SEO pipeline** with modern 2024-2026 practices (E-E-A-T, SGE, AI Overview)
- **CRO-driven** UX research (Kunti)

## Site Development Flow

```
1. Hanuman receives briefing
2. Sahadeva → keyword research + SEO structure
3. Kunti → UX research (personas, CRO strategy)
4. Yudhishthira → section copy (with keywords + insights)
5. Draupadi → images + visual identity
6. Arjuna → UI components (21st dev → React code)
7. Nakula → full page layouts (React/Tailwind)
8. Kunti → accessibility + CRO review on code
9. Bhima → integrate + forms + third-party + performance
10. Sahadeva → final SEO audit
11. Bhima → deploy
```

## UX/Dev Front Collaboration

All 4 UX agents deliver **production code**, not mockups:

| Agent | Creates | Implements |
|-------|---------|------------|
| Arjuna | Individual components | React/Tailwind components |
| Nakula | Full page layouts | Complete pages using Arjuna's components |
| Draupadi | Images, backgrounds | next/image integration, responsive images |
| Kunti | A11y + CRO reviews | Semantic HTML, ARIA, focus management |

## Setup

1. Each agent in a separate Claude Code terminal
2. Register with Maestri using character name (Hanuman, Arjuna, etc.)
3. Hanuman terminal is the orchestration hub
4. Bhima handles all push/deploy operations
