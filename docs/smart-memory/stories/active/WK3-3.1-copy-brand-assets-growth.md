# Story WK3-3.1: Copiar brand assets Growth

## Objetivo
Disponibilizar os ativos visuais da Growth Sales dentro do projeto (`/public/brand/growth/`) para que os slides do workshop-3 possam referenciá-los via URL pública.

## Acceptance Criteria
- [ ] AC1: Diretório `public/brand/growth/` criado
- [ ] AC2: Copiados todos os arquivos de `/Users/joaoramos/Downloads/growth-sales-a-i-manual-da-marca/project/refundacao/assets/` para `public/brand/growth/` preservando nomes e estrutura (incluindo subdir `imagery/`)
- [ ] AC3: Verificar que ao menos os seguintes arquivos estão presentes em `public/brand/growth/`: `favicon.png`, `favicon.svg`, `logo-header.png`, `og-image.png`, `symbol-official.svg`, e o diretório `imagery/`
- [ ] AC4: Nenhum arquivo binário foi corrompido durante a cópia (verificar com `file` ou abrir SVG para checar conteúdo válido)
- [ ] AC5: Nenhum arquivo fora desse diretório de origem foi movido/copiado/alterado
- [ ] AC6: Listagem `ls public/brand/growth/` reportada no File List da story após conclusão

## Escopo

**IN:**
- Cópia simples dos assets para `public/brand/growth/`
- Preservar nomes originais e subdiretórios

**OUT:**
- Otimização de imagens (resize, compressão, conversão para webp)
- Renomeação ou normalização de nomes
- Atualização do `favicon.ico` global do site (workshop-3 é noindex, não muda o branding do domínio)
- Adicionar ao `git` (decisão fica para o lead após validação)
