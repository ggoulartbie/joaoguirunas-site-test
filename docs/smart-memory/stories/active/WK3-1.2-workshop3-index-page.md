# Story WK3-1.2: Página índice /workshop-3

## Objetivo
Página `/workshop-3` que lista os 9 slides com timing, marca Growth Sales, e serve de hub de navegação durante a apresentação ao vivo.

## Acceptance Criteria
- [ ] AC1: Arquivo `src/app/workshop-3/page.tsx` criado, exportando `Workshop3IndexPage` default
- [ ] AC2: Importa `WORKSHOP_3_SLIDES` de `./_components/Workshop3DeckLayout`
- [ ] AC3: Renderiza header com label "Deck · {total} min · 9 slides", título grande "Growth Sales — Encerramento" e subtítulo curto sobre o evento
- [ ] AC4: Renderiza lista (`<ol>`) clicável com os 9 slides em grid `sm:grid-cols-2`, cada item mostra `Slide NN`, duração, título
- [ ] AC5: Hover de cada item destaca borda + título com accent Growth
- [ ] AC6: `metadata` exporta `title: 'Growth Sales — Encerramento'`, `robots: { index: false, follow: false }` (noindex para evento privado), `alternates.canonical: '/workshop-3'`
- [ ] AC7: `export const dynamic = 'force-dynamic'` no topo (igual workshop-2)
- [ ] AC8: Layout responsivo: `max-w-5xl`, padding generoso
- [ ] AC9: Identidade visual Growth (background, foreground, accent) lidos do brand md do ux

## Escopo

**IN:**
- Página `/workshop-3` apenas
- Listagem dos 9 slides com link e duration
- Header com título do evento + subtítulo
- noindex via metadata

**OUT:**
- Conteúdo dos slides individuais
- Layout (vem do WK3-1.1)
- Section "Stack" como o workshop-2 tem — workshop-3 é deck, não tutorial
- CMS/auth/gating

## Contexto Técnico

Referência: `src/app/workshop-2/page.tsx`. Estrutura igual, sem a seção `STACK`.

Cálculo de `totalMinutes`: como duration tem formato `2:30`, fazer parse de minutos+segundos para mostrar algo como "20 min" no header. Aceitável também mostrar literalmente "20 min" hardcoded já que a soma é fixa (2+2+2+2.5+2.5+2+1.5+2.5+3 = 20).
