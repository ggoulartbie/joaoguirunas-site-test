# Story F10.3: /curso-online — landing page + CTA na home

## Objetivo
Criar a página `/curso-online` como landing page de venda do Curso Online | Claude Agents Team (R$ 499), com CTA directo para Stripe Checkout, e adicionar botão de acesso na home.

## Produto
- **Nome:** Curso Online | Claude Agents Team
- **Preço:** R$ 499,00
- **Diferencial vs Mentoria:** NÃO inclui encontros em grupo ao vivo NEM os bônus de frameworks aceleradores (código dos sistemas Growth Sales)
- **Inclui:** acesso completo ao conteúdo em vídeo, materiais, fórum, certificado

## Acceptance Criteria

- [ ] AC1: Página `/curso-online` criada com design consistente com `/mentoria` (mesma linguagem visual KV)
- [ ] AC2: Secções obrigatórias: hero com proposta de valor, o que está incluído, o que NÃO está incluído (diferencial honesto), depoimentos/prova social (placeholder se não houver), preço + CTA
- [ ] AC3: CTA principal "Comprar agora — R$ 499" chama `createCheckoutSession` com o slug do cohort online (verificar slug real na DB ou usar `curso-online-padrao`)
- [ ] AC4: Página funciona sem login (pública) — o checkout lida com auth
- [ ] AC5: Home (`/`) — adicionar botão/card "Curso Online" no hero ou secção de produtos que leva a `/curso-online`
- [ ] AC6: Meta tags (title, description, OG) específicas para `/curso-online`
- [ ] AC7: Link `/curso-online` adicionado ao sitemap (`src/app/sitemap.ts`)

## Escopo
**IN:** nova page `/curso-online`, CTA na home, sitemap
**OUT:** checkout Stripe em si (já existe em F7), conteúdo real de copywriting (usar placeholder estruturado)
