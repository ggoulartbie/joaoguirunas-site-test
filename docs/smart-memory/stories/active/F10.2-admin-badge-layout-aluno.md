# Story F10.2: Badge Admin no layout do aluno

## Objetivo
Adicionar um botão/badge discreto "⚙ Admin" no header da área de aluno, visível apenas para utilizadores com `role = 'ADMIN'`, que permite saltar para `/academy/admin` a qualquer momento.

## Acceptance Criteria

- [ ] AC1: Header da área aluno — botão "⚙ Admin" ou "Admin ↗" visível APENAS quando `profile.role === 'ADMIN'`
- [ ] AC2: Botão abre `/academy/admin` (nova tab ou mesma tab — preferência: mesma tab)
- [ ] AC3: Design discreto — não perturba a experiência de aluno (ex: ícone pequeno + texto no canto superior direito, junto ao avatar/perfil)
- [ ] AC4: Utilizadores STUDENT e MENTOR não vêem o botão (nem no DOM)
- [ ] AC5: O badge funciona em mobile (não quebra o layout do header)

## Escopo
**IN:** layout do aluno (`src/app/(academy)/academy/(student)/layout.tsx` ou header component)
**OUT:** protecção de rotas admin (já existe via middleware), criação de alunos (F10.4)
