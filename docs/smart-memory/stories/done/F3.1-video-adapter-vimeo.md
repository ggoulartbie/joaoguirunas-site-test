# Story F3.1: Video adapter + Vimeo implementation

## Objetivo

Criar abstração de vídeo (`/lib/video/`) com interface neutra e implementação Vimeo, permitindo trocar provider depois sem mexer em UI.

## Acceptance Criteria

- [x] AC1: `lib/video/interface.ts` define `VideoAdapter` interface com `getEmbedUrl(videoId, options)` e `getVideoMetadata(videoId)`. `uploadVideo` não implementado na interface — considerado out of scope (Vimeo upload é feito manualmente no painel)
- [x] AC2: `lib/video/vimeo.ts` implementa Vimeo com domain-locking via `VIMEO_DOMAIN_WHITELIST` env var
- [x] AC3: `getEmbedUrl()` retorna URL com domain-locking e parâmetros de privacidade
- [x] AC4: `import 'server-only'` presente em `vimeo.ts` — token Vimeo nunca exposto no client
- [x] AC5: `lib/video/index.ts` exporta factory `getVideoAdapter(provider)` e `getVideoAdapterForLesson(lesson)`
- [ ] AC6: Variáveis de ambiente documentadas no README — não verificado (README pode não existir ou estar desatualizado)
