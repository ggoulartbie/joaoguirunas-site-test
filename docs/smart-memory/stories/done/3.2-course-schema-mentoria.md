# Story 3.2: Course schema JSON-LD em /mentoria

## Objetivo
Adicionar `Course` schema JSON-LD em `/mentoria` para aumentar elegibilidade a rich results no SERP. A página de mentoria é o funil principal de conversão e não tinha nenhum schema personalizado.

## Acceptance Criteria

- [x] AC1: `src/app/mentoria/page.tsx` injeta `<script type="application/ld+json">` com schema `@type: "Course"`.
- [x] AC2: Schema inclui `name`, `description`, `provider` (GrowthSales.ai), `inLanguage`, `url`, e `offers` com `price`, `priceCurrency`, `availability`.
- [x] AC3: `tsc --noEmit` passa sem erros.
- [ ] AC4: Validar em Google Rich Results Test — sem erros críticos.
