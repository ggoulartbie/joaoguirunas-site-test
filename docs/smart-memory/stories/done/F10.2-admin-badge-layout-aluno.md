# Story F10.2: Badge Admin no layout do aluno

## Objetivo
Adicionar um botão/badge discreto "⚙ Admin" no header da área de aluno, visível apenas para utilizadores com `role = 'ADMIN'`, que permite saltar para `/academy/admin` a qualquer momento.

## Acceptance Criteria

- [x] AC1: Header da área aluno — botão com ícone Settings + texto "Admin" visível APENAS quando `profile.role === 'ADMIN'`
- [x] AC2: Botão navega para `/academy/admin` (mesma tab)
- [x] AC3: Design discreto — borda `var(--hairline)`, cor `var(--bone-mute)`, font-mono 10px. Não interfere com layout existente
- [x] AC4: Utilizadores STUDENT/MENTOR não vêem o botão — renderização condicional no Server Component; prop `isAdmin` não passa ao DOM
- [x] AC5: Mobile: ícone visível em todos os tamanhos; label "Admin" ocultado em xs com `hidden sm:inline`
