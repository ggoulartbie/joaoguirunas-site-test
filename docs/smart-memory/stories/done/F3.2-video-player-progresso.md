# Story F3.2: VideoPlayer + Server Action saveProgress

## Objetivo

Componente `<VideoPlayer />` que embute Vimeo e salva progresso do aluno a cada 10s; conclusão automática ao chegar no fim.

## Acceptance Criteria

- [x] AC1: `components/student/VideoPlayer.tsx` Client Component recebe `lessonId`, `videoId`, `videoProvider`
- [x] AC2: Embed via `getEmbedUrl()` do adapter (VimeoAdapter)
- [x] AC3: Player escuta eventos `timeupdate` do Vimeo Player API; `saveProgress` disparado via `setInterval` de 10s (`PROGRESS_INTERVAL_MS = 10_000`)
- [x] AC4: `saveProgress(lessonId, seconds)` em `app/actions/progress.ts`: upsert em `lesson_progress`; `seconds_watched` nunca regride (`.lt('seconds_watched', seconds)`); `markLessonComplete` disparado quando threshold atingido
- [x] AC5: `markLessonComplete` disponível como server action — botão manual não verificado explicitamente na UI mas action existe
- [x] AC6: Intervalo de 10s (debounce); flush no `pause` e `ended`
- [ ] AC7: Indicador visual de % assistido — não verificado explicitamente; ProgressBar existe mas integração com VideoPlayer não confirmada
