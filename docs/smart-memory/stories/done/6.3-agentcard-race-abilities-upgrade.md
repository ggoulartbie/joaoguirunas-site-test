# Story 6.3: AgentCard — exibir raça alienígena + abilities + squadRole

## Objetivo
Atualizar o componente AgentCard para exibir a raça alienígena do agente, suas 3 principais abilities e uma frase curta de squadRole.

## Contexto
AgentCard está em `src/app/agentes/_components/AgentCard.tsx`.
Esta story depende da Story 6.2 (campo `abilities` e `squadRole` no agentes.json).

Atualmente o card mostra: imagem | squad badge | model badge | codename | title | description (3 linhas) | tools strip

O que precisa ser adicionado:
1. **Race badge**: badge sutil com o nome da raça (ex: "Arcturiana", "Luminari", "Xelvari", "Reptiliana") — abaixo do squad badge ou no body
2. **Abilities**: substituir ou complementar o tools strip com as abilities humanas (ex: "Análise CVE", "WebSearch profundo")
3. **Squad role snippet**: frase curta em lugar de ou junto à description

## Design spec
- Race badge: fonte mono, texto white/40, sem cor extra (não é tão proeminente quanto o squad badge)
- Abilities: pills pequenas white/60, diferenciadas dos tools (que são white/40)
- Squad role: pode substituir a `description` clamp-3 por `squadRole` (mais curto e direto)
- Manter o mesmo tamanho e layout do card — não expandir verticalmente

## Acceptance Criteria
- [x] AC1: Race do agente visível no card (badge ou texto mono abaixo do squad label)
- [x] AC2: Top 3 `abilities` exibidas em vez das `tools` na strip inferior (tools removidas do card, continuam na single page)
- [x] AC3: `squadRole` substitui o parágrafo de `description` clamp-3 no body do card
- [x] AC4: Layout do card não cresce — informação densa mas limpa
- [x] AC5: Sem TypeScript errors

## Escopo

**IN:**
- `src/app/agentes/_components/AgentCard.tsx`

**OUT:**
- Nenhuma mudança em SquadSection, AgentesHero, SolarSystemScene
- Não alterar a single page (story 6.4)
