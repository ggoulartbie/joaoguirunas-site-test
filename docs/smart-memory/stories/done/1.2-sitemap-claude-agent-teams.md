# Story 1.2: Adicionar /skills/claude-agent-teams ao sitemap.xml

## Objetivo
Incluir a rota `/skills/claude-agent-teams` no `src/app/sitemap.ts` para que seja indexada por buscadores e apareça em `sitemap.xml`.

## Acceptance Criteria
- [x] AC1: `src/app/sitemap.ts` contém entrada `{ path: '/skills/claude-agent-teams', priority: 0.7, freq: 'monthly' }` no bloco `// Skills` (prioridade elevada para 0.7 dado relevância ao core do site).
- [ ] AC2: Após build, a URL `https://joaoguirunas.com/skills/claude-agent-teams` aparece em `/sitemap.xml`.
- [x] AC3: Contagem total de URLs atualizada de 46 para 47.
- [ ] AC4: Documentação `docs/smart-memory/project/modules.md` §5 atualizada.
- [x] AC5: Página `/skills/claude-agent-teams` permanece intacta.

## Escopo

**IN:**
- Editar `src/app/sitemap.ts` — adicionar linha no array `pages[]` dentro do bloco `// Skills`.
- Atualizar tabela em `docs/smart-memory/project/modules.md` §5.
- Atualizar número total de URLs em qualquer doc que cite "47 URLs" se necessário.

**OUT:**
- Criação/edição da página `/skills/claude-agent-teams`.
- Automatização de descoberta de rotas para sitemap (seria outra story, complexidade L — ver riscos em [[../../project/overview]] §Pontos de atenção item 2).
- Re-priorização de outras entradas do sitemap.

## Contexto Técnico

**Evidência do gap:**
- `ls src/app/skills/` lista 24 diretórios incluindo `claude-agent-teams`.
- `src/app/sitemap.ts:32-55` lista 22 skills — `claude-agent-teams` ausente.
- Portanto página existe, é estática e linkável, mas não aparece no sitemap.xml gerado.

**Prioridade sugerida:** `0.6` (alinhada com skills mais relevantes como `copywriting`, `website-builder`, `crm`). Justificativa: tema alinhado com a proposta Claude Code + squads, core da autoridade do site. Pode ser elevada em story futura se analytics mostrarem tráfego/intenção de busca forte.

**Frequência:** `monthly` (mesmo padrão das demais skills).

**Constraints:**
- Sitemap é hardcoded (ver [[../../project/overview]] §Pontos de atenção 2). Story atual mantém esse padrão — refactor pra autodiscovery é fora de escopo.
- Deploy na Vercel regenera sitemap automaticamente no próximo build.
