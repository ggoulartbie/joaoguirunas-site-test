# Story 4.1: KV Global — tokens de cor, dot-grid texture e watermark diamante

## Objetivo
Aplicar os 3 tokens globais do KV Growth Sales que afetam todas as páginas: refinamento do orange token, dot-grid texture como CSS global, e o componente GrowthWatermark (diamante SVG ghosted).

## Acceptance Criteria
- [x] AC1: `--color-accent` refinado de `#FF4400` → `#FF3A0E` em `globals.css` (+ hover `#FF5A1F`)
- [x] AC2: Background do `body` ajustado de `#08080C` → `#050507` (void) — verificar que não quebra cards/surfaces
- [x] AC3: Dot-grid texture adicionada como `body::before` em `globals.css`: `radial-gradient(rgba(255,255,255,0.04) 1px, transparent 1px) / 24px 24px`, `opacity: 0.55`, `pointer-events: none`, `position: fixed`, `z-index: 0`
- [x] AC4: Componente `<GrowthWatermark />` criado em `src/shared/components/ui/growth-watermark.tsx` — SVG do diamante GS com `opacity: 0.05`, `position: absolute`, `pointer-events: none`, `aria-hidden`
- [x] AC5: `GrowthWatermark` instanciado na Home (hero section) e na Mentoria (hero section) — tamanho `600px`, posicionado canto inferior direito fora do flow
- [x] AC6: Verificar contraste de texto: orange `#FF3A0E` sobre `#050507` passa WCAG AA (relação ≥ 4.5:1 para texto small)
