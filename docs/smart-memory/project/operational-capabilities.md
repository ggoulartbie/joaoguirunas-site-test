# Capacidades Operacionais

## Supabase CLI

- Binário: `/opt/homebrew/bin/supabase` (v2.90.0)
- Projeto linkado: `mksmmpfyqowuzjcchhkl` (João Guirunas | Academy)
- Comando SQL direto: `supabase db query --linked "SQL"`
- Migrations: `supabase db push` (aplica em produção)
- Types: `supabase gen types typescript --linked`

## Regra de delegação

O lead (team-os) nunca executa operações diretamente.
Toda operação de banco, arquivo ou deploy é delegada aos teammates.
