## Dress Code — Estilo Casual (Conteúdo Creator)

### Camisetas internas (variar entre as 3)
| Cor | Tom | Uso |
|-----|-----|-----|
| **Laranja vibrante** | ~#E8601C | Padrão histórico — mais energia, mais vibrante |
| **Preta** | #1A1A1A | Mais sério, editorial, contrasta com casaco floral |
| **Roxa** | ~#6B21A8 | Criativa, diferente, boa com plantas ao fundo |

> Variar entre as 3 a cada geração — não repetir sempre laranja.

### Camada externa (casaco floral — SEMPRE variado)
- **Bomber jacket floral** — preta com flores tropicais coloridas (hibiscus, flores grandes, vibrantes)
- **Camisa floral aberta** — preta ou escura com flores variadas, usada aberta sobre a camiseta
- Padrões florais devem variar entre gerações — nunca o mesmo padrão duas vezes seguidas
- Referência visual: `Casual/02-multiplas-telas.png`, `Casual/casual-camisa-floral-sorrindo-corredor.png`

### Calças
- **Jeans escura** — padrão. Às vezes com detalhes/destroyed.

### O que NUNCA usar (casual)
- Terno ou blazer formal (reservado para conteúdo de autoridade corporativa — outro contexto)
- Roupas lisas sem estampa floral na camada externa
- Looks all-white ou all-beige

## Estilo de Locais — Cenários

### Cenário Principal: **Escritório Biofílico**
O ambiente que mais representa o João. Características obrigatórias:

| Elemento | Detalhe |
|----------|---------|
| **Plantas** | Muitas — tropicais, densas. Monstera, palmeiras, arbustos verdes grandes. Sempre presentes, não apenas decorativas |
| **Iluminação** | Warm amber — Edison bulbs, pendentes rústicos, cordões de luz. Nunca luz fria/fluorescente |
| **Janelas** | Grandes, com luz natural entrando. Madeira nas molduras ou teto |
| **Teto** | Estrutura de madeira exposta ou vigas aparentes |
| **Tech** | Múltiplas telas/monitores ao fundo (com conteúdo colorido, dashboards, interfaces) |
| **Vibe** | Creative studio / startup — não corporativo, não formal |

### Cenário Secundário: **Co-working moderno com plantas**
- Espaço amplo, open space
- Luz natural lateral abundante
- Plantas em vasos grandes pelos corredores
- Arquitetura contemporânea mas aconchegante

### Cenário Ocasional: **Exterior urbano / varanda**
- Entradas de prédios modernos com vegetação
- Varandas com plantas
- Sempre com toque de verde

### O que NUNCA usar como cenário
- Escritório corporativo padrão (sem plantas, luz fria, mesa genérica)
- Fundo branco/liso sem textura
- Ambientes extremamente formais (sala de reunião clássica)
- Cenários sem nenhuma planta

## Estratégia de Personagem por Formato

### Capas de Feed / Reels (4:5 ou 9:16)
Duas opções válidas — variar conforme o contexto:

**Opção A — Clone do João (Freepik)**
- Usar clone `@joaoguirnas` no Freepik AI
- Dress code: camiseta [preta/laranja/roxa] + casaco floral variado
- Cenário: escritório com plantas

**Opção B — Atores com diversidade**
- Usar atores com raça, cor e etnia variadas (não restrito ao João)
- Representatividade intencional: negro, asiático, latino, médio-oriental, branco — variar
- Mesma lógica de dress code casual criativo e cenário biofílico
- Útil para conteúdo mais universal ou quando o clone não está disponível

### Carrosséis (slides internos)
- **Formato obrigatório: Ultrawide Cinematic 21:9** (`film_horizontal_21_9` no Freepik)
- Personagens variados (Opção B preferida para slides — maior diversidade visual)
- Cenário: sempre escritório/espaço com plantas
- Estilo: ultra realista, fotográfico, sem ilustração
